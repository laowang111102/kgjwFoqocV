# Python招聘大数据可视化分析系统 python670

## 项目概述

本项目是一款基于Python语言的招聘大数据可视化分析系统，旨在为用户提供直观的招聘信息和薪资数据展示，通过数据分析和预测，帮助用户更好地理解市场需求和薪资趋势。

## 技术栈

- 后端框架：Flask
- 数据库：MySQL
- 前端技术：HTML, CSS, JavaScript（具体框架未提及）

## 功能模块

### 1. 首页数据展示

- 岗位信息数量
- 行业平均薪资
- 行业最低/最高薪资
- 学历与薪资统计
- 整体学历要求
- 福利待遇
- 热门岗位
- 数据的可视化展示

### 2. 数据概况

- 开发岗位分类
- 非开发岗位分类
- 薪资预测
- 岗位匹配
- 支持按学历要求、职位进行精准搜索

### 3. 可视化

- **薪资情况**：通过学历筛选，展示不同岗位的薪资分布和比例，使用柱状图和饼图
- **企业情况**：查看职位的招聘城市、公司规模，以及主要城市的职位比例，使用饼图展示
- **学历情况**：展示不同职位的学历和工作经验要求，使用漏斗图和矩形树图

### 4. 智能化

- **薪资预测**：根据职位、学历、公司规模、城市、工作经验等条件，选择不同模型预测最低薪资
- **职位匹配**：通过相似条件，筛选出最符合用户要求的职位信息

## 系统角色

- 用户：查看数据、进行职位搜索、薪资预测等
